# お問合せ動作確認チェックリスト

## 概要

本ドキュメントは、AIコードレビュー学習サービスのお問い合わせフォーム機能の動作確認を体系的に行うためのチェックリストです。

**対象機能**: お問い合わせフォーム（`/contact`）  
**技術スタック**: Next.js 15 (App Router), MUI, MongoDB, Mongoose, Nodemailer  
**確認環境**: デスクトップ（1280px以上）、タブレット（768-1279px）、スマートフォン（767px以下）

## 参考情報

### MongoDBコレクション構造

**コレクション名**: `contacts`

**スキーマ**:
```typescript
{
  _id: ObjectId,
  name: String (必須, 最大100文字),
  email: String (必須, 最大254文字, 小文字変換),
  subject: String (必須, 最大200文字),
  message: String (必須, 10-5000文字),
  status: String (enum: "unread" | "read" | "replied", デフォルト: "unread"),
  createdAt: Date (自動生成)
}
```

**インデックス**:
- `createdAt`: 降順（最新順取得用）
- `status`: 未読フィルタリング用
- `email`: ユーザー別問い合わせ検索用
